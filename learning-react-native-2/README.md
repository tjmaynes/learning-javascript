## Requirements

- [Nodejs](https://nodejs.org)
- [Xcode](https://developer.apple.com/xcode/)

## Usage

To install dependencies, run the following command:
```bash
npm test
```

To develop the iOS app, run the following command:
```bash
npm run ios
```

To run tests, run the following command:
```bash
npm test
```
